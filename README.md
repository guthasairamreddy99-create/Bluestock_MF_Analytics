# 📊 Mutual Fund Analytics Capstone Project

## 📌 Project Overview

This project analyzes Mutual Fund datasets using Python, Pandas, SQLite, and SQL. It includes data ingestion, data cleaning, database creation, and analytical SQL queries to generate meaningful insights from mutual fund data.

---

## 🚀 Features

- Data Ingestion from CSV files
- Data Cleaning and Preprocessing
- SQLite Database Creation
- SQL Schema Design
- Analytical SQL Queries
- Data Dictionary Documentation
- GitHub Version Control

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- NumPy
- SQLAlchemy
- SQLite
- Git & GitHub

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/guthasairamreddy99-create/Bluestock_MF_Analytics.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run Data Ingestion

```bash
python data_ingestion.py
```

### Run Data Cleaning

```bash
python data_cleaning.py
```

### Create SQLite Database

```bash
python sql_database.py
```
